My project is titled: Repair-it.
It is a progressive web application, that aimed at people in previously disadvantaged backgrounds such as townships,informal settlements and rural areas.
My aim with this project is to connect house holds, with verified, quality semi-skilled artisans for all thier home repairs and to connect semi-skilled artisans with clients.
This project can be used for learning purposes because I'm also learning,by using it I hope you will also improve your programming skills and logic.
Technologies Used: HTML, Css and JavaScript.
Anyone with knowledge or learning like I am,can use and contribute to this project.
I also hope to improve writing README.md files as I continue to learn.
